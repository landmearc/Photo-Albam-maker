# Photo-Albam-maker
Pythonを使って、画像ファイルからフォトアルバムを自動作成するツールです。
主として境界写真帳を作成するツールです。
iphoneなどで撮影された写真はデータが大きくそのまま写真帳を作ろうとすると巨大なデータになってしまいます。
A4サイズの用紙に遠景、近景の2まいを各測点毎に写真を撮影していることがほとんどなので、1ページ当たり4枚の写真と説明文の構成になるようにレイアウトします。
合わせて写真も640×480のVGAサイズにリサイズを行います。
測量図のPDFファイル又は画地データのSIMAファイルを読み込み、プレビュー画面上で点名をダブルクリックすることで写真のファイル名をリネームすることが出来ます。
また、新設、既設、復元、その他の区分やコメントも追記できます。
